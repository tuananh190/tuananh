            2. CSS selector
                * Bộ chọn phần tử 
                tagHTML{...} 
                ex: h1{...} p{...}
                *Bộ chọn ID
                # ID {...}
                  ex: #zone1{back-ground-color:#ccff12}
                * Bộ chọn Class
                .class-name {...}
                ex:
                   .noi-dung{
                    color:blue
                   }
                *Bộ chọn theo giá trị thuộc tính
                tag[attribute=value]{.....}
                ex:
                   input[type="text"]{
                        border: 1px soild #ccc;
                   }
                *Bộ chọn ngữ cảnh
                    tag:hover 
                    tag:active
                    tag:before / tag:after
                    tag:first-child ; ... 
                    ...
                    ex:
                       a: hover{
                        color:red
                       }
                *Bộ chọn:
                    - tag01 , tag02{...}
                    - tag01 > tag02{...}
                    - tag01 ~ tag02{...}
                    - tag01 + tag02{...}
                    ....
